# Problem: Reverse String

<br>

Given a string `s` reverse the string.

<br>

**Example 1:**

<pre>
<b>Input:</b> s = "Hello"
<b>Output:</b> "olleH"
</pre>

**Example 2:**

<pre>
<b>Input:</b> s = "Hello World!"
<b>Output:</b> "!dlroW olleH"
</pre>

**Example 3:**

<pre>
<b>Input:</b> s = "12345"
<b>Output:</b> "54321"
</pre>

<br>

**Note**: **CANNOT USE STRINGBUILDER!**
